Hello user!!

This is an application for checking the permissions your apps are taking from you,
so that you can know what permission is doing what and events like 
"Calculator" asking for calling permission may not happen anymore!!

- Instructions

1. download the code from the repository.
2. open you command line interface.
3. navigate to the directory having the application.
4. Type "python main.py" in the interface.
5. If it shows any error, check that you have the python version greater than version "3.2".


!!Have fun!!
